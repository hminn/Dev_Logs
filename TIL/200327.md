      ```python
      # Room admin list에서 등록된 room의 명칭이 'Room Object(1)' 이런 식으로 되어있다. (string method를 따로 구현해주지 않았다면)
      
      # 만약 list에 room에 설정한 room_name을 표시하고 싶다면 다음과 같이 string method를 구현해주면 된다.
      
      def __str__(self):
          return self.name
      ```
